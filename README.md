# cgmsharp

## Install dependencies

```
npm install
```

Or

```
npm i @diakem/libre-link-up-api-client
npm i @babel/runtime
npm i dotenv
```

## Configure environment variables

```
$ touch .env
LIBRE_USERNAME=email
LIBRE_PASSWORD=secret
LIBRE_VERSION=4.9.0
```

## Call LibreLinkUp API Client

```
node readCgmData.js
```
