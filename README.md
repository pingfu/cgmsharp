# cgmsharp



## Install dependencies

```
npm install
```

Or

```
npm i github:marcbarry/libre-link-up-api-client
npm i @babel/runtime
npm i dotenv
```

## Configure environment variables

```
$ touch .env
LIBRE_USERNAME=email
LIBRE_PASSWORD=secret
LIBRE_VERSION=4.9.0
PUSHOVER_USER=secret
PUSHOVER_TOKEN=secret
```

## Run gcm-monitor

```
node readCgmData.js
```
