# cgmsharp

## Install dependencies

```
npm install
yarn install
```

Or

```
npm i @diakem/libre-link-up-api-client
npm i @babel/runtime
npm i dotenv
```

## Configure environment variables

```
$ touch .env
LIBRE_USERNAME=email
LIBRE_PASSWORD=secret
```

## Call LibreLinkUp Api Client

```
node readCgmData.js
```
